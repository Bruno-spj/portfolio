Pokédex

Projeto desenvolvido com React, TypeScript, Vite, HTML5 e CSS, que oferece uma listagem interativa e dinâmica dos Pokémons. A aplicação consome a API Pokémon para exibir informações detalhadas sobre cada Pokémon, incluindo nome, imagem, tipo e habilidades. Com uma interface moderna e responsiva, a Podédex permite aos usuários explorar e pesquisar seus Pokémons favoritos de forma prática e eficiente.

API - movieFlix

Este foi meu primeiro projeto acadêmico no curso de programação backend. Desenvolvi uma API RESTful capaz de adicionar, buscar e deletar filmes de um banco de dados.
